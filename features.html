Features

  • Lists available armor sets in game
  • Displays materials required to complete armor upgrades
  • Click set name to collapse/expand
  • Checkbox to include/exclude individual sets from item count
  • Right click or long hold to select/deselect all checkboxes
  • Click each item to indicate if you have completed that upgrade (tick)
  • Right click any item to tick/untick whole set
  • Count updates based on what sets you are tracking and what items you have already upgraded
  • Items and quantities to finish upgrading your sets indicated at the bottom of the page
  • Hover over an item in this list to see what sets it is needed to finish
  • Click an item in this list to see the items wiki entry
  • Set completion/inclusion saved between visits using local storage.
